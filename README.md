## nginx  
  
nginx is a shell designed for interactive use  
  
Automatic install/update:

```shell
bash -c "$(curl -LSs https://github.com/systemmgr/nginx/raw/master/install.sh)"
```

Manual install:
  
requires:

Debian based:

```shell
apt install nginx
```  

Fedora Based:

```shell
yum install nginx
```  

Arch Based:

```shell
pacman -S nginx
```  

MacOS:  

```shell
brew install nginx
```
